# 💸 Personal Expense Analyzer (Java Edition)
**Author:** Asmit Upadhyay

## 📌 Problem Statement
Students and young professionals often lose track of their daily minor expenses. Without a clear, frictionless system to log where money is going, it becomes difficult to stick to a budget.

## 💡 Solution
The Personal Expense Analyzer is a lightweight Java Desktop Application that allows users to quickly log their daily transactions, categorize them, and instantly view their total spending. It requires no complex database setup, making it perfectly portable.

## 🚀 Features
* **Frictionless Entry:** Quickly add expenses with titles, categories, and amounts via a clean GUI.
* **Persistent Storage:** Automatically saves all transactions locally to a `.csv` file so data is never lost between sessions.
* **Dynamic Analytics:** Automatically calculates and updates the total amount spent in real-time.

## 🛠️ Tech Stack
* **Language:** Core Java (JDK 21)
* **GUI Framework:** Java Swing & AWT
* **Data Storage:** Flat File storage (CSV)

## ⚙️ Local Setup & Installation

1. **Clone the repository**
   git clone https://github.com/Pubaso200/java-expense-tracker.git
   cd java-expense-tracker

2. **Compile the Application**
   javac ExpenseTracker.java

3. **Run the Application**
   java ExpenseTracker
